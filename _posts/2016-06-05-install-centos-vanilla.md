1. On you server, download CentOS ISO file in a directory of your choice  

~~~bash  
wget http://isoredirect.centos.org/centos/7/isos/x86_64/CentOS-7-x86_64-DVD-1611.iso .
~~~

2. Start the KVM console  

~~~bash  
virt-manager
~~~
